# 爬虫能力与数据管理分离 — 实施计划

> **For Claude:** REQUIRED SUB-SKILL: Use superpowers:executing-plans to implement this plan task-by-task.

**目标：** 将 MediaCrawler-Api 单体服务拆分为独立的 Crawler Service (端口 8081) 和 Data API Service (端口 8080)，共享 MySQL 实例。

**架构：** 提取 shared/ 共享包（ORM + DB Session），Crawler Service 负责爬虫执行和数据写入，Data API Service 负责数据查询/AI/关键词/平台管理和前端静态文件。前端直连两个服务。

**技术栈：** Python 3.12+, FastAPI, SQLAlchemy, Playwright, React + Vite, pnpm, MySQL 8.0

```python
"""Crawler Service — FastAPI 应用入口"""
import asyncio
from contextlib import asynccontextmanager
from fastapi import FastAPI
from fastapi.middleware.cors import CORSMiddleware

from api.routers.crawler import crawler_router
from api.routers.websocket import websocket_router
from api.services.crawler_manager import crawler_manager


@asynccontextmanager
async def lifespan(app: FastAPI):
    # 启动后台日志广播任务
    broadcast_task = asyncio.create_task(crawler_manager._broadcast_loop())
    monitor_task = asyncio.create_task(crawler_manager._monitor_process())
    yield
    broadcast_task.cancel()
    monitor_task.cancel()


app = FastAPI(title="MediaCrawler Crawler Service", version="1.0.0", lifespan=lifespan)

app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:5173", "http://localhost:3000", "http://127.0.0.1:5173", "http://127.0.0.1:3000", "http://localhost:8080"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)

app.include_router(crawler_router, prefix="/api")
app.include_router(websocket_router, prefix="/api")


@app.get("/api/health")
async def health():
    return {"status": "ok", "service": "crawler-service"}
```

### Task 5.2：前端 request.ts 支持多 baseURL

**文件：**

**Step 1：修改 request.ts**

将单例 axios 改为两个实例：

```typescript
import axios, { AxiosRequestConfig } from 'axios';
import { message } from 'antd';

// Data API Service (默认) — 端口 8080
const apiClient = axios.create({
  baseURL: import.meta.env.VITE_API_BASE_URL || '',
  timeout: 60000,
});

// Crawler Service — 端口 8081
const crawlerClient = axios.create({
  baseURL: import.meta.env.VITE_CRAWLER_BASE_URL || 'http://localhost:8081',
  timeout: 60000,
});

// 错误拦截器工厂
function setupErrorInterceptor(client: typeof apiClient) {
  client.interceptors.response.use(
    (response) => response,
    (error) => {
      const detail = error.response?.data?.detail || error.response?.data?.message;
      if (detail) message.error(detail);
      return Promise.reject(error);
    }
  );
}

setupErrorInterceptor(apiClient);
setupErrorInterceptor(crawlerClient);

// 根据 URL 前缀选择客户端
// /api/crawler/* 和 /api/ws/* → crawlerClient
// 其余 → apiClient
function getClient(url: string) {
  if (url.startsWith('/api/crawler') || url.startsWith('/api/ws')) {
    return crawlerClient;
  }
  return apiClient;
}

export async function httpGet<T>(url: string, config?: AxiosRequestConfig) {
  const client = getClient(url);
  const response = await client.get<T>(url, config);
  return response.data;
}

export async function httpPost<T>(url: string, data?: any, config?: AxiosRequestConfig) {
  const client = getClient(url);
  const response = await client.post<T>(url, data, config);
  return response.data;
}

export async function httpDelete<T>(url: string, config?: AxiosRequestConfig) {
  const client = getClient(url);
  const response = await client.delete<T>(url, config);
  return response.data;
}
```
